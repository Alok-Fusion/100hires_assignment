# Setup Log: Cursor, Claude Code, and Codex

This repository documents the first step of the 100Hires application process, where I installed and configured a set of AI coding tools.

## What I installed

I installed Cursor IDE from cursor.com. Once Cursor was set up, I went into the Extensions panel and searched for Claude Code, installed it, and logged in. I did the same for the Codex add-on, searching for it in Extensions, installing it, and signing in.

## Steps I completed

1. Downloaded and installed Cursor IDE.
2. Opened Cursor and went to the Extensions tab.
3. Searched for "Claude Code," installed the extension, and logged in with my account.
4. Searched for "Codex" in the same Extensions tab, installed it, and logged in.
5. Created this public GitHub repository under my existing GitHub account.
6. Opened the repository directly inside Cursor.
7. Wrote this README to document the process.
8. Committed and pushed the changes to GitHub.



I already had a GitHub account from earlier projects, so the only new part here was creating a fresh repository for this task. Installing Cursor itself was straightforward, just a normal download and install. Finding both extensions inside Cursor's Extensions panel took a moment since I wasn't sure exactly what to search for at first, but searching "Claude Code" and "Codex" directly brought them up without much trouble. Logging into both add-ons was a standard sign-in flow once I found them.

Overall the setup went smoothly. Nothing here was complicated on its own, it was mostly about being methodical and going through each tool one at a time rather than rushing.
